Standards

Filenames: Use all lowercase in file and folder names.

Filenames: Use index.html as your default home page names.

Use dashes or underscores instead of spaces

Your GitHub repository must be named wdd130. I will make a video demonstrating how to fix this if your repository is not correctly named.

Semantic vs. non-semantic tags

The div and span tags are not semantic. They are containers only and don't give any hint about what's in them.

Three cases where non-semantic tags are useful: Styling and layout only, JavaScript manipulation, Custom Widgets
